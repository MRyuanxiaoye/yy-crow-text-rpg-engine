# 黑乌鸦 — 语感校准参考

> 从五篇用户修改稿（图穷、沈园、白茶、浣溪沙、浮生）中提炼的实际句式。
> 本文档的用途是写作前校准语感——读一遍，找到"对的感觉"，然后放下。
> 不是模板库，不要从里面抠句式往自己的文字上套。如果某个句子拿掉主语后放到任何文章里都成立，那它大概率是模板，需要重写。

---

## 一、开头方式

### 1.1 地点+感官细节起手（不解释，不判断）

- "燕国的冬天有一种气味，是烧不尽的枯草和冻土混在一起的那种，闷在鼻腔里" — 图穷
- "苏州城西有条巷子，窄，两个人并排走要侧身" — 浮生
- "康熙十二年的北京，入秋以后风就硬了。从西山那边刮过来，带着沙" — 浣溪沙
- "杭州的雨分很多种。春天的雨是细的，落在西湖上像有人在水面上写字" — 白茶
- "绍兴的春天是从墙根底下开始的。青苔先绿，然后是檐下的燕子回来" — 沈园

**规律**：地名开头，紧跟一个具体的感官细节（气味、触感、声音、视觉），不做"X是Y的"判断句，不做"不是A而是B"的否定-肯定对仗。

### 1.2 人物+一句话交代处境

- "荆轲在这种气味里住了三年" — 图穷
- "赵大勇在这条河边蹲了快两个月" — 第一章改
- "苏晚在里开了三年茶馆" — 白茶
- "纳兰成德坐在书房里，手边摊着一卷词稿，墨干了也没蘸" — 浣溪沙

**规律**：人名+动作/状态+时间跨度，一句话完成人物出场和处境交代，不补充解释。

### 1.3 章节/段落切换——事件起手，不用范式过渡

- "田光来找他的那天，下着大雪" — 图穷
- "樊於期是自己来找荆轲的" — 图穷
- "唐婉嫁过来的那年，春天来得早" — 沈园
- "十年后，沈园" — 沈园
- "秋天的时候出了一件事" — 白茶
- "乾隆四十年，沈复十八岁，芸娘嫁过来了" — 浮生
- "嘉庆八年三月三十日。芸娘死在扬州" — 浮生
- "康熙十三年五月三十日。卢氏生产" — 浣溪沙

**规律**：时间/事件直接起手，不用"他是被X的"、"X开始了"、"接下来发生的事改变了一切"等范式。日期+事件是最干净的切换方式。

## 二、描写方式

### 2.1 人物外貌——挑一两个细节，不做全身扫描

- "四十来岁，颧骨高，下巴上一撮没刮干净的胡茬。眼睛不大，盯人的时候很专注" — 第一章
- "圆脸，很白，眉毛淡得像是用水画的" — 浣溪沙
- "她瘦，头发用一根布条绑着，袖口磨出了毛边" — 浮生
- "来人高，瘦，外套上沾着雪粒子，头发湿了贴在额头上" — 白茶
- "逃亡的日子把一个将军磨成了一个老头，眼窝深陷，颧骨撑着一层干皮。但眼睛还是亮的" — 图穷

**规律**：2-3个视觉细节，用逗号连接，不用"他的X是Y的，他的Z是W的"逐项列举。最后一个细节往往带性格暗示。

### 2.2 环境——一句白描定调，不铺陈

- "那天晚上没有月亮" — 图穷
- "窗外什么都看不清，玻璃上全是水雾" — 白茶
- "阳光从巷子口斜过来，照在那几片叶子上，亮了一下" — 白茶
- "窗外有鸟叫，一声一声的，隔着窗纸传进来，闷闷的" — 浣溪沙

**规律**：一句话，一个感官通道，不叠加多个意象。环境描写是节奏的呼吸点，不是展示文笔的地方。

### 2.3 动作——动词串联，不加修饰

- "他伸出手，五根指头张开，又一根一根收回去" — 图穷
- "她咬断一根线，把针别在衣襟上，揉了揉眼睛" — 浮生
- "芸娘收拾了一个包袱，不多，几件换洗衣裳，一本书" — 浮生
- "他从袖子里摸出一截炭笔，在墙上写了一首词" — 沈园
- "她拿了一块，咬了一口" — 白茶

**规律**：动词之间用逗号连接，不用"然后"、"接着"、"随后"等连接词。不加副词（"缓缓地"、"轻轻地"）。

## 三、情感表达

### 3.1 用物件和动作传情，不用对话直说

- "手炉每天都在，炭火每天都是新换的，这件事就这么存在着，像呼吸一样" — 浣溪沙
- "芸娘在他折角的地方又折了一次，变成了双折" — 浮生
- "粥还是热的，上面卧了一个荷包蛋" — 浮生
- "苏晚没叫住他，把多的钱压在收银台的玻璃板底下" — 白茶
- "她的手在他掌心里松了" — 浣溪沙

**规律**：关键情感节点不写内心独白，不写"他感到X"，只写一个动作或一个物件的状态。

### 3.2 克制的悲伤——越重的事越轻地写

- "沈复等着她说下一句。过了很久，沈复没有等到芸娘的下一句" — 浮生
- "还是没砸中。瞎了，毕竟是瞎了" — 图穷
- "她死的那天，绍兴下了一场雨。跟她离开陆家那天一样，是那种沾衣服的毛毛雨" — 沈园
- "秦舞阳死在了殿外。据说他到最后也没能把嘴闭上" — 图穷

**规律**：不渲染，不铺垫情绪，用事实本身的重量传递悲伤。句子越短，力度越大。

### 3.3 感慨——极少使用，一篇最多1-2处

- "是啊，花不会嫌弃花瓶，嫌弃花瓶的从来都是人" — 浮生
- "好的日子就是这样，过的时候不觉得，回头看才知道，每一天都是在借" — 浮生
- "有些事，做不成也得做。有些路，走不通也得走" — 图穷

**规律**：只在重要的气口、节奏转折点上用，画龙点睛，不能多。

## 四、对话方式

### 4.1 对话极简，靠动作补充语境

- "什么时候要？" "不急。" "我急。" — 图穷
- "薄荷呢？" "死了。" — 白茶
- "看见了吗？" "看见了。像你。" — 浣溪沙
- "手炉呢？" "给你用了。" — 浣溪沙
- "来世。" — 浮生

**规律**：对话不超过一两句，不在对话中灌输信息。双方都知道的事不说出来。

### 4.2 对话后的动作用逗号连接，不断开

- "太子是个好人。"樊於期在他对面蹲下来，膝盖咔嚓响了一声，"好人做不了大事。" — 图穷
- "你外婆说得对，"她说，"白茶就是靠等。" — 白茶
- "我急。"樊於期说，"多活一天就多欠一天。早点了了，干净。" — 图穷

## 五、过渡与收尾

### 5.1 场景结尾——留一个未完成的动作或意象

- "石榴树在风里抖了一下，掉了几片叶子" — 图穷
- "那个酒窝没有出来" — 沈园
- "茶还是温的" — 白茶
- "他放在了手炉旁边，像是给谁看" — 浣溪沙

**规律**：不总结，不点题，不写"他知道一切都变了"。用一个具体的物件或细节收束，让读者自己体会。

### 5.2 时间跳跃——直接跳，不铺垫

- "十年后，沈园" — 沈园
- "后来陆游活到八十五岁" — 沈园
- "芸娘死后，沈复开始写书" — 浮生
- "卢氏死后，纳兰把她缝到一半的那件衣裳收进了箱子里" — 浣溪沙

**规律**：大时间跳跃用一句话完成，不用"时光飞逝"、"日子一天天过去"等过渡句。

## 六、禁止句式清单（AI高频模板）

| 禁止句式 | 替代方式 |
|----------|---------|
| "他是被X的" | 直接写动作："一瓢凉水兜头泼下来" |
| "X是Y的。不是那种A的Y，是B的Y" | 只写一个具体细节，不做否定-肯定对仗 |
| "不是A，是B"（同篇超过1次） | 换其他句式，或直接写B |
| "像是X，又像是Y"对仗式双重比喻 | 只保留一个比喻 |
| "然后/接着/随后"作为段落开头 | 直接写下一个动作 |
| "他感到X"直接陈述情感 | 写身体反应或动作 |
| "这一刻，他知道了X" | 删掉，让读者自己判断 |
| "X得不像Y的" | 换成具体描述 |
| 段落末尾总结句（"他知道一切都变了"） | 用物件/细节收束 |
| "说不上是X还是Y" | 只写一个，或都不写 |

---

## 使用方式

1. 每次写作前精读本文档，校准句式感觉
2. 写完一段后回头检查：是否有禁止句式清单中的模板
3. 如果某个句子拿掉主语后放到任何文章里都成立，那它大概率是模板，需要重写
4. 同一种句式结构在一篇文章里不重复出现超过一次
