# 全书架构

## 基本信息

- 类型/题材：玄幻（悬疑解谜主线 + 渡亡单元剧 + 修炼升级 + 势力博弈）
- 预计总字数：200万+
- 预计总章数：~600章（每章3000-4000字）
- 卷数：9卷

## 世界观

详见 `novels/001_诡秘之主/indexes/worldbuilding_改.md`，此处仅列核心要素：

- 神魔乱世，道佛并存博弈，百家散落
- 12境界体系（前9凡俗 + 后3圣境），道佛双命名
- 30条秘传（道8 + 佛8 + 百家14）+ 佛道融合秘传（主角专属）
- 特殊空间：天界（道）、净土（佛）、洞天福地（百家）、地府（归属不明）
- 天书十九卷：远古大能记录的起源真相，残卷散落各方
- 主角金手指：玉坠（地府归属碎片）→ 遗物空间系统 → 渡亡经（佛道融合秘传）

## 卷级结构

### 第一卷：浮尸渡口
- 章节范围：第1章 - 第60章
- 核心冲突：浮尸案调查 — 谁在用活人献祭开地府之门？
- 开卷状态：主角被捞起，失忆，被看管，一无所有
- 结卷状态：浮尸案真相揭开，主角初步掌握渡亡经，离开渡口镇，进入更大的世界
- 关键转折点：
  1. 第一次被动进入遗物空间（发现能力）
  2. 第一次主动渡亡，获得愿力反馈，解锁渡亡经
  3. 发现浮尸案背后是献祭开门
  4. 第一卷高潮：献祭现场正面冲突，主角用渡亡经的能力逆转局面
  5. 道佛两方势力介入，主角被迫选择站位（选择都不站）
- 主角境界：无 → 守静/持戒（一境）
- 节奏：快节奏，每2-3章一个小爽点，抓读者

### 第二卷：行走江湖
- 章节范围：第61章 - 第130章
- 核心冲突：主角离开渡口镇后，在江湖中行走，遇到各种死者和活人的纠葛，逐步建立自己的渡亡方式
- 开卷状态：初入江湖，有基础修为但实力弱，靠渡亡积累愿力
- 结卷状态：在江湖中有了一定名声（好坏参半），遗物空间扩展，收留了几个重要的亡魂角色
- 关键转折点：
  1. 遇到第一个不愿被渡的亡魂（渡亡不是万能的）
  2. 遗物空间第一次扩展，空间内出现变化
  3. 遇到佛道正统势力的人，被发现修炼的功法"不正统"
  4. 第一次遇到同样能接触亡魂的人（竞争者/同行）
  5. 卷末：发现某个亡魂的遗愿牵出一个大势力的秘密
- 主角境界：一境 → 三境（含德/忍辱）
- 节奏：单元剧为主，每个渡亡案件2-5章，穿插主线线索

### 第三卷：入局
- 章节范围：第131章 - 第200章
- 核心冲突：主角被卷入道佛势力的博弈，不得不在夹缝中求生
- 开卷状态：因为第二卷末的发现，被大势力盯上
- 结卷状态：在势力博弈中站稳脚跟，获得一定的自主权，但也树敌
- 关键转折点：
  1. 被某一方势力"邀请"（半强迫）加入
  2. 在势力内部发现地府相关的线索
  3. 渡亡经突破关键节点，解锁新能力（能看到活人身上的因果线）
  4. 发现天书十九卷的第一条线索
  5. 卷末：势力内部叛变/大事件，主角被迫暴露部分实力
- 主角境界：三境 → 五境（坐忘/禅定）
- 节奏：开始多线并进，势力线+渡亡线+天书线交织

### 第四卷：地府初窥
- 章节范围：第201章 - 第270章
- 核心冲突：主角第一次真正接触地府，发现地府崩塌的真相
- 开卷状态：实力中等，有了自己的小团队和亡魂伙伴
- 结卷状态：亲眼见到地府残骸，理解了地府无主的含义，决定承担修补地府的责任
- 关键转折点：
  1. 遗物空间与地府产生共鸣，出现通道
  2. 第一次进入地府残骸，看到崩塌的景象
  3. 发现地府崩塌与远古大能有关
  4. 获得天书十九卷的一份残卷
  5. 卷末：地府中的某种存在苏醒，主角被迫撤离，但带出了关键信息
- 主角境界：五境 → 六境（抱一/般若）— 质变，可称高手
- 节奏：探索+悬疑为主，节奏略慢但信息密度高

### 第五卷：天下乱
- 章节范围：第271章 - 第350章
- 核心冲突：地府异变的消息泄露，天下大乱，各方势力争夺地府入口
- 开卷状态：主角已是高手，但在大势力面前仍不够看
- 结卷状态：在乱局中保全自己和同伴，获得了修补地府的关键方法
- 关键转折点：
  1. 地府异变引发天灾（亡魂外泄、阴气侵蚀），普通人受害
  2. 主角用渡亡经大规模渡亡，名声大振但也成为众矢之的
  3. 道佛两方各自的地府方案曝光（都想独占）
  4. 主角发现渡亡经与地府的真正关系——渡亡经就是地府的运行法则
  5. 卷末：一场大战，主角被重创，同伴有人牺牲
- 主角境界：六境 → 七境（玄览/观照）
- 节奏：战争+乱世节奏，紧张度持续拉高

### 第六卷：独行
- 章节范围：第351章 - 第420章
- 核心冲突：主角在重创后独自修行，重新审视自己的道路
- 开卷状态：重创，同伴离散，低谷
- 结卷状态：突破瓶颈，对渡亡经有了全新理解，重新出发
- 关键转折点：
  1. 独自流浪，遇到底层百姓的苦难，回归渡亡初心
  2. 在一个偏僻之地发现远古大能的修行遗迹
  3. 获得天书十九卷的更多残卷，拼凑出部分真相
  4. 渡亡经大成，理解"渡人即渡己"
  5. 卷末：收到消息，同伴有难，重新踏入江湖
- 主角境界：七境 → 八境（虚极/寂灭）
- 节奏：慢节奏，内省+修行+偶发事件，情感线为主

### 第七卷：破局
- 章节范围：第421章 - 第500章
- 核心冲突：主角回归，以全新的实力和理解介入天下格局
- 开卷状态：八境高手，渡亡经大成，有了明确的目标
- 结卷状态：打破道佛对地府的争夺僵局，提出第三条路
- 关键转折点：
  1. 回归后救出同伴，展示实力飞跃（升级验证）
  2. 与道佛双方的顶尖人物正面交锋
  3. 揭露天书十九卷的核心秘密——地府本就不该属于道或佛
  4. 突破九境（归根/涅槃），凡俗之巅
  5. 卷末：主角宣布要重建地府，道佛联手施压
- 主角境界：八境 → 九境（归根/涅槃）— 凡俗之巅
- 节奏：快节奏，大场面，高潮频繁

### 第八卷：证道
- 章节范围：第501章 - 第560章
- 核心冲突：主角突破圣境的过程，面对道佛两方的全面围剿
- 开卷状态：九境巅峰，准备突破圣境
- 结卷状态：突破十境，初次感知气运/因果，获得地府的部分掌控权
- 关键转折点：
  1. 突破圣境需要巨大的变故作为契机
  2. 道佛联手围杀，主角的重要之人死亡（巨大变故）
  3. 在绝境中突破十境，渡亡经与地府法则融合
  4. 获得地府部分掌控权，但地府仍不完整
  5. 卷末：远古大能留下的最终考验浮现
- 主角境界：九境 → 十境（得一/罗汉）— 入圣
- 节奏：压抑→爆发，全书最大的情感高潮

### 第九卷：地府真神
- 章节范围：第561章 - 第600章+
- 核心冲突：主角完成最终考验，成为地府真神
- 开卷状态：十境，地府部分掌控权，但远古真相未解
- 结卷状态：成为第四位真神，地府重建，天下新秩序
- 关键转折点：
  1. 天书十九卷全部集齐，远古真相揭开
  2. 发现远古大能的真正身份和他毁灭地府的原因
  3. 最终考验：不是战斗，而是选择——是否愿意承担地府的重量
  4. 主角做出选择，渡亡经圆满，成为地府真神
  5. 尾声：道主、佛主、圣人承认第四位真神，天下格局重塑
- 主角境界：十境 → 十二境（合道/佛）— 但走的是第三条路
- 节奏：收束，回收所有伏笔，温情结尾
