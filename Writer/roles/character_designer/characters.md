# 人物档案总表

## 主角

### 陈渡
- 身份/职业：穿越者，原身是被献祭的普通人，穿越前是现代程序员
- 外貌特征：普通，不起眼，瘦，河里泡过之后脸色发白，后来慢慢恢复
- 性格/习惯：心思缜密，观察力强，不爱说话但脑子一直在转。善良但不圣母，帮人有底线。穿越前的程序员思维让他习惯拆解问题、找规律。
- 立场/利益：活下去，弄清楚自己身上发生了什么，不想被任何势力控制
- 当前认知状态（第7章后）：知道玉坠能感知亡魂、能进入灰色空间跟死者对话、完成遗愿能获得愿力。刚获得渡亡经，体内有气在走，但不知道这是什么功法，不知道自己修炼的是秘传。不知道浮尸案的真相，只知道河里不断有人死。
- 弧光轨迹：被动承受 → 主动选择渡亡 → 在渡亡中找到自己的道 → 承担地府的重量 → 成为真神
- 出场章节：第1章起，全书
- 存活状态：活着

## 核心配角

### 方平川
- 身份/职业：清平帮管事，渡口镇实际管事人之一
- 外貌特征：三十出头，精瘦，眼睛很亮，笑起来不像好人
- 性格/习惯：精明，两面三刀，说话绕弯子，从不直接表态。习惯背着手走路。
- 立场/利益：清平帮的利益优先，但他自己也有私心。知道浮尸案的部分内情，在各方势力之间周旋，想从中获利。底线不明。
- 与主角的关系：看管者/试探者。对主角有兴趣，但不是善意的兴趣——他在评估主角有没有利用价值。
- 当前认知状态（第7章后）：知道主角是唯一存活的浮尸，怀疑主角身上有特殊之处。知道河底有阵法（知道多少待定）。不知道主角能渡亡。
- 弧光轨迹：精明的投机者 → 在大势面前做出选择 → 结局待定（第一卷末揭晓）
- 出场章节：第2章起
- 存活状态：活着

### 僧人（法号待定）
- 身份/职业：佛家某寺派来调查浮尸案的僧人
- 外貌特征：高大，光头，面相年轻但眼神老成，穿灰色僧袍，脚上的草鞋磨得很薄
- 性格/习惯：话少，但说出来的每句都有分量。不吃肉但不劝别人。走路很轻，存在感可以随意收放。
- 立场/利益：佛家的任务是调查浮尸案，但他个人对地府之事有自己的看法。对主角身上的气息感到好奇——既不是道也不是佛，但又两者都沾。
- 与主角的关系：后期合作者。第一个察觉主角修炼的功法不寻常的人。
- 当前认知状态：知道浮尸案不简单，知道道家也派了人来，知道清平帮在里面有角色。
- 弧光轨迹：执行任务的僧人 → 被主角的渡亡方式触动 → 成为主角的重要盟友
- 出场章节：第21章起（预计）
- 存活状态：活着

### 徐道士
- 身份/职业：道家某派弟子，被派来调查浮尸案，死去的道袍年轻人的师兄
- 外貌特征：二十多岁，瘦高，脸上有一道旧疤（小时候练剑留的），头发用木簪束着，道袍洗得发白
- 性格/习惯：急性子，嘴快，但心不坏。师弟死了之后变得沉默了很多。喝酒，但酒量不好。
- 立场/利益：为师弟报仇，查清浮尸案真相。道家的任务和个人仇恨重合。
- 与主角的关系：主角传递了他师弟的遗言，这份恩情让他信任主角。后期的战斗搭档。
- 当前认知状态：知道师弟发现了河底的秘密后被杀。通过主角传递的消息知道河底有阵法。
- 弧光轨迹：为师弟复仇的道士 → 在调查中发现更大的真相 → 成长为独当一面的人
- 出场章节：第16章起（预计）
- 存活状态：活着
