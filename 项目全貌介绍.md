# YY-Crow Tools - 项目全貌介绍

> AI 驱动的智能工具集平台，包含深度搜索和世界观构建两大核心工具

---

## 项目概述

这是一个采用**微服务架构**的 AI 应用平台，使用 Docker 容器化部署，通过 Nginx 反向代理统一对外提供服务。项目包含两个核心工具：

| 工具 | 功能定位 | 技术特色 |
|------|----------|----------|
| **深度搜索** | 多引擎聚合的深度知识检索系统 | 实体感知规划、分层搜索、混合阅读器 |
| **世界观编织器** | 小说/游戏世界观构建工具 | 双模型协作、图谱式管理、多模式 Agent |

## 技术栈

### 前端
- **框架**: Next.js 14/16 + React 18/19 + TypeScript
- **UI**: Tailwind CSS + Lucide React + Framer Motion
- **特色**: ReactFlow (思维导图)、React Markdown、SSE 流式输出

### 后端
- **框架**: FastAPI + Uvicorn
- **AI 框架**: LangChain + LangGraph (工作流编排)
- **大模型**: GPT-4o (逻辑校验) + DeepSeek (创意生成)

### 搜索引擎集成
- Tavily Search (主力)
- Exa API (深度搜索)
- Bocha API (中文增强)
- DuckDuckGo (备用)

### 数据处理
- ChromaDB (向量数据库)
- BM25 + Jieba (中文检索)
- KeyBERT (关键词提取)
- Supabase (云端缓存)

## 核心功能详解

### 一、深度搜索工具

**工作流程 (LangGraph 状态图)**:

```
check_cache → planner → researcher → reader → writer
   缓存检查     智能规划    并发搜索    深度阅读   报告生成
```

**核心特性**:

1. **实体感知规划**
   - 自动识别核心实体类型（人物/事件/概念/事物/组织）
   - 根据实体类型动态选择分析框架
   - 框架融合：主框架深度挖掘 + 次框架浅层覆盖

2. **分层搜索策略**
   - Primary 查询：宏观 + 2-3 个深度子查询
   - Secondary 查询：仅 1 个浅层查询
   - 避免信息过载，提高效率

3. **混合阅读器**
   ```
   Level 1: 爬虫抓取原文
   Level 2: BM25 关键词初筛
   Level 3: 向量语义精筛
   Level 4: LLM 深度总结（仅高相关内容）
   ```

4. **瀑布式写作**
   - 层层深入，不设上限
   - 因果链条连贯
   - 精彩故事突出概括

5. **云端知识库缓存**
   - Supabase 存储历史查询
   - 命中缓存秒级返回

---

### 二、世界观编织器

**数据模型**:
- **WorldNode**: 12 种节点类型（世界/地区/角色/魔法/历史等）
- **NodeRelation**: 7 种关系类型（包含/属于/依赖/冲突等）
- **WorldGraph**: 完整世界图谱

**多 Agent 协作架构**:

```
                    build_context (上下文构建)
                           │
           ┌───────────────┼───────────────┐
           │               │               │
           ▼               ▼               ▼
    ┌─────────────┐ ┌─────────────┐ ┌─────────────┐
    │   Creator   │ │  Architect  │ │  Character  │
    │  (DeepSeek) │ │   (GPT-4o)  │ │  (DeepSeek) │
    │  查漏补缺    │ │  逻辑校验    │ │  角色分析    │
    └─────────────┘ └─────────────┘ └─────────────┘
```

**对话模式**:
| 模式 | Agent | 功能 |
|------|-------|------|
| audit | Creator + Architect | 查漏补缺 + 冲突检测 |
| validate | Architect | 科学/玄学依据校验 |
| character | Character | 设定完整度 + 事件关联分析 |
| chat | Creator | 自由对话（带历史记忆） |

## 运行方式

### 开发模式（支持热重载）

```bash
# 首次启动（构建镜像）
docker compose -f docker-compose.dev.yml up --build

# 日常开发
docker compose -f docker-compose.dev.yml up

# 后台运行
docker compose -f docker-compose.dev.yml up -d

# 停止服务
docker compose -f docker-compose.dev.yml down

- Hub 首页: http://localhost:3001

docker compose up --build -d

# 查看日志
docker compose logs -f

# 停止服务
docker compose down
```
