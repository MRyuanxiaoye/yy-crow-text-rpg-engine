# 高级分析方向总览

> 本文档记录了项目中提出的所有分析方向、优先级排序及实现状态。

## 背景

现有方法链（stage0-4）的所有分析都是**按号码编号**进行的，完全没有利用大乐透前区红球 `a<b<c<d<e` 升序排列带来的**位置信息**。

关键发现：
- 红球总是严格升序，每个位置有隐含范围（位置1:[1,31], 位置2:[2,32], ..., 位置5:[5,35]）
- 相邻位置 Pearson 相关系数 0.65-0.73
- 现有 stage0 没有按位置分析的特征提取

---

## 10个分析方向

### 方向1：位置序列模式挖掘 + 差分分析

- 同位置相邻N期的升降模式
- 对每个位置，将相邻期的变化编码为方向序列：U(升), D(降), E(平)
- 窗口大小从3到8逐步扫描
- 统计每种模式的频率，与随机期望做卡方检验
- 差分序列分析：差分绝对值分布、正负交替节奏、二阶差分
- 当前已知最近N-1期方向，匹配历史前缀预测下一个方向
- 自动搜索每个位置的最优窗口大小（回测准确率）

**输出**：
- `position_predictions`: 每个位置的预测值域 + 方向 + 置信度
- `direction_patterns`: 每个位置的显著模式列表
- `diff_analysis`: 差分统计（一阶/二阶）
- `optimal_windows`: 每个位置的最优窗口大小

### 方向2：差分序列分析

不看号码本身，看**相邻期的差值**。比如第1位置连续5期是 `3, 7, 5, 12, 8`，差分序列是 `+4, -2, +7, -4`。

- 差分的绝对值是否有分布规律（比如大跳后倾向小跳）
- 差分的正负交替是否有节奏
- 二阶差分（差分的差分）是否趋近于0（类似物理中的加速度衰减）

> 注：在实际实现中，方向1和方向2合并为一个模块 `position_pattern.py`。

### 方向3：拓扑数据分析（TDA）

把每期号码看作高维空间中的一个点（5维向量），用**持续同调（Persistent Homology）**分析点云的拓扑结构：

- 号码序列是否形成环状结构（周期性）
- 是否存在"空洞"（某些号码组合区域从不出现）
- 拓扑特征随时间的变化（滑动窗口TDA）
- 这能发现传统统计方法看不到的**形状特征**

### 方向4：信息论方法

用熵和互信息量化号码序列的"可预测性"：

- **条件熵**：已知前N期，第N+1期的不确定性有多大？N越大熵越低说明规律越强
- **互信息**：不同位置之间的信息共享量（第1位置的变化能预测第3位置多少）
- **传递熵**：方向性的因果关系（是第1位置"驱动"第3位置，还是反过来）
- 自动找出信息量最大的特征组合

### 方向5：符号动力学

把号码序列转化为符号序列，用动力系统理论分析：

- **符号化**：将号码空间分区（小/中/大），每期变成一个符号串如 `SMLMS`
- **Lyapunov指数**：衡量序列的混沌程度（完全随机 vs 有确定性成分）
- **递归图（Recurrence Plot）**：可视化序列中的重复模式，自动检测周期和状态转换
- 如果Lyapunov指数显示序列不是完全随机的，就说明存在可利用的结构

### 方向6：序列模式挖掘（SPM）

数据挖掘领域的经典方法，不预设模式长度和形状：

- **PrefixSpan/GSP算法**：自动发现频繁子序列
- 不只是升降，而是发现类似"第1位置出小号 → 2期后第3位置出大号"这种跨位置、跨时间的关联规则
- 支持gap约束（模式中间可以间隔几期）
- 输出：所有统计显著的模式及其支持度和置信度

### 方向7：时间序列形态匹配（Shapelet）

- **Shapelet**：自动从历史中提取最具区分力的"子形状"
- 比如发现"V形反转"（降-降-升-升）这个形状出现后，下一期大概率继续升
- 不需要人工定义形状，算法自动搜索
- 用DTW（动态时间规整）做形态相似度匹配，容忍时间轴上的伸缩

### 方向8：因果推断

超越相关性，寻找因果关系：

- **Granger因果检验**：第1位置的历史值是否能帮助预测第3位置的未来值
- **收敛交叉映射（CCM）**：适用于非线性系统的因果检测
- 如果发现某些位置之间存在因果关系，就可以用"因"来预测"果"

### 方向9：隐马尔可夫模型（HMM）

假设号码序列背后存在**不可观测的隐藏状态**：

- 比如"热态"、"冷态"、"过渡态"
- 算法自动学习：有几个隐藏状态、状态之间的转移概率、每个状态下的号码分布
- 当前处于什么状态 → 预测下一期的号码分布
- 状态数量也可以自动搜索（BIC准则）

### 方向10：多尺度分析（小波变换）

FFT只能看全局频率，小波变换能看**局部频率随时间的变化**：

- 某个号码可能前500期有7期的周期，后500期变成了12期的周期
- 小波变换能捕捉这种"周期本身在变化"的现象
- 输出时频图，直观看到每个号码在不同时间段的周期性强弱

## 优先级排序

| 优先级 | 方向 | 理由 |
|--------|------|------|
| **P0** | 位置序列模式 + 差分分析（方向1+2） | 最直接，实现简单，效果可验证 |
| **P0** | 序列模式挖掘 SPM（方向6） | 不预设规律，自动发现，正是需要的 |
| **P0** | 信息论评估（方向4） | 先回答"数据中到底有多少可利用的信息"，避免在完全随机的数据上白费力气 |
| **P1** | Shapelet形态匹配（方向7） | 发现"形状"规律，与走势图分析直接对应 |
| **P1** | 隐马尔可夫模型 HMM（方向9） | 发现隐藏状态，理论基础扎实 |
| **P1** | 信息论方法（方向4扩展） | 先用来评估"到底有多少可预测性"，指导其他方法 |
| **P2** | 小波变换（方向10） | 增强现有FFT，发现变化的周期 |
| **P2** | 拓扑数据分析 TDA（方向3） | 发现高维结构，但实现复杂 |
| **P2** | 因果推断（方向8） | 发现位置间因果关系 |
| **P3** | 符号动力学（方向5） | 理论价值高，但实用性需验证 |
