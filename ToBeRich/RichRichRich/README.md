# 彩票数据分析推理项目

## 项目简介

这是一个基于时序数据分析的推理项目，以中国大乐透和双色球的历史开奖数据为研究对象，探索数据分析和推理方法在时序数据中的应用。

**重要声明**：本项目是一个数据分析学习项目，不是彩票预测系统。彩票开奖结果是随机的，本项目仅用于数据分析方法的学习和实践。

## 项目特性

### 1. 数据驱动的分析框架
- 基于真实历史开奖数据
- 时序数据特征提取
- 多维度统计分析

### 2. 双层推理机制

#### 第一层：排除法推理
通过分析历史数据规律，推理出下一期不太可能出现的号码：
- **连续性规律**：分析号码的连续出现模式
- **周期性规律**：识别号码出现的周期特征
- **分布规律**：基于号码分布特征进行排除
- **极值规律**：排除极端组合（如全奇、全偶、全大、全小等）

#### 第二层：权重计算
对未被排除的号码计算出现权重：
- **历史频率权重**：基于号码的历史出现频率
- **时序衰减权重**：近期数据赋予更高权重
- **间隔期权重**：基于号码未出现的间隔期数
- **组合特征权重**：考虑号码组合的历史表现

### 3. 组合优化策略
- 将排除法和权重法结合
- 生成多组候选号码组合
- 评估组合的合理性和多样性

### 4. 可视化分析
- 号码频率分布图
- 时序趋势分析图
- 冷热号码分析
- 间隔期分布图

## 技术架构

### 核心技术栈
- **Python 3.8+**：主要开发语言
- **pandas**：数据处理和分析
- **numpy**：数值计算
- **matplotlib/seaborn**：数据可视化
- **scipy**：统计分析

### 模块设计
```
分析引擎
├── 数据加载模块：读取和验证历史数据
├── 统计分析模块：基础统计和特征提取
├── 排除推理模块：基于规律的号码排除
├── 权重计算模块：计算号码出现权重
├── 组合生成模块：生成候选号码组合
└── 可视化模块：结果展示和报告生成
```

## 分析维度

### 大乐透分析
- **前区**：从1-35中选5个号码
- **后区**：从1-12中选2个号码
- 分析维度：
  - 号码频率分析
  - 奇偶比例分析
  - 大小比例分析
  - 区间分布分析
  - 连号分析
  - 和值分析

### 双色球分析
- **红球**：从1-33中选6个号码
- **蓝球**：从1-16中选1个号码
- 分析维度：
  - 号码频率分析
  - 奇偶比例分析
  - 大小比例分析
  - 三区分布分析
  - 连号分析
  - 和值分析

## 评估方法

本项目通过以下指标评估分析方法的有效性：

1. **排除准确率**：被排除的号码在实际开奖中未出现的比例
2. **权重相关性**：高权重号码的实际出现频率
3. **组合覆盖率**：生成的候选组合中包含实际中奖号码的比例
4. **多样性指标**：生成组合的多样性和合理性
