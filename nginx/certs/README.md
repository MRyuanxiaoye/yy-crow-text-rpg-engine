# SSL 证书文件说明

请将从阿里云下载的 SSL 证书文件放置在此目录，并**重命名**为以下文件名：

- `yy-crow.com.pem` - SSL 证书文件
- `yy-crow.com.key` - SSL 私钥文件

## 示例

如果你从阿里云下载的文件名是：
- `1234567_yy-crow.com.pem`
- `1234567_yy-crow.com.key`

请将它们重命名为：
- `yy-crow.com.pem`
- `yy-crow.com.key`

## 注意

- 确保这两个文件都存在
- 文件权限应该是私有的（不要提交到 Git）
- 部署前务必检查文件名是否正确
